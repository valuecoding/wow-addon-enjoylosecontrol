# EnjoyLoseControl

`EnjoyLoseControl` is a TBC Anniversary fork of `LoseControl`, maintained by `enjoymygripz`.

Version: `1.0.0`

## What It Does

The addon shows crowd control and interrupt durations as dedicated icons on supported unit frames.

## Supported Units

- player
- pet
- target
- targettarget
- focus
- focustarget
- party
- arena
- raid
- nameplate

## TBC Anniversary Work Included

- updated options compatibility for TBC Anniversary
- improved arena anchor handling
- cleaned up slider visuals in the config UI
- added an Info panel with support and CurseForge contact links
- added select-to-copy link fields for support/contact URLs
